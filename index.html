Because the wheel doesn't need reinventing

What is it?

  • A way to boostrap your AEM project with commonly required functionality
  • A set of reusable components
  • An AEM development toolkit

What isn't it?

  • A framework
  • A library which has required third party dependencies
  • A library which changes standard AEM functionality by default

How do I use it?

  • Add it as a sub package using Maven details
  • Manually deploy the package using the package manager

Where do I get it?

  • From repo.adobe.com
  • Github Releases

Who are you?

We are Adobe Consulting Services. We build solutions on top of Adobe products including Adobe Experience Manager (AEM, fka CQ, fka Communique).

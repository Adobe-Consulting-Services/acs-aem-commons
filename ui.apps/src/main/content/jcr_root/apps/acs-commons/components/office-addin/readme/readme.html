AEM Asset Selector Office Add-in

Setup Instructions

  • Create a page in your instance and add the 'Asset Selector' component
    • Usable sample provided at /content/office-addin/asset-selector
  • In the component dialog you can change the root path if you want to point to specific folder
  • Copy the URL to this page
  • Update the manifest files to use this URL.
    • At line 19 change DefaultValue of SourceLocation to use this URL
    • At line 71 change the DefaultValue of Taskpane.Url to use this URL
  • Now the add-ins are ready to install by following these directions to make the addin available in your organization.
